Detection: DNS-based exfiltration is notoriously difficult to detect. Lambda functions do not send DNS query logs to CloudTrail by default. Route 53 Resolver Query Logs must be explicitly enabled on the VPC (not available for non-VPC Lambda). For non-VPC functions, DNS queries leave no AWS-native log trail at all — detection requires network-layer controls outside AWS. TCP reverse shells, by contrast, are detectable via VPC Flow Logs (if in a VPC) or GuardDuty's network anomaly findings. MITRE: T1048.001.
